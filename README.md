# HR Analytics Dashboard ‚Äì Power BI

This project is an end-to-end HR Analytics dashboard built using **Power BI**, focusing on **Employee Attrition** and **Employee Performance**.  
It uses the Kaggle HR dataset and covers complete workflow: data cleaning, modeling, DAX calculations, KPI creation, and interactive reporting.
